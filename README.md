# boing ✈️

*Never miss a connection. Boing notifies you when you're in the same city as your friends.*

## What is boing?

`boing` is a Flutter-based mobile app designed for frequent travelers who want an easier way to stay connected to their travel connections.

The app keeps track of the city you are in and automatically sends you a notification if you happen to be in the same city as one of your connections. It's built for travelers who want to serendipitously boing between friends while overseas.

## Core Features

* **Automatic City Detection:** Uses background location services to always know what city you're in.
* **Friend Connection System:** Easily add friends to your network via QR code scanning.
* **Serendipitous Notifications:** Get a push notification when you and a friend are in the same city at the same time.
* **Secure User Accounts:** Powered by Firebase for authentication (including Google Sign-in) and secure data storage.

## Tech Stack

This project is built with Flutter and relies on a Firebase backend.

* **Framework:** Flutter
* **Backend:** Firebase (Authentication, Cloud Firestore, Firebase Storage)
